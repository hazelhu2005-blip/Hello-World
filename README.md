# Float-CET6-Word-Memorizer
桌面悬浮PDF背单词工具（Python Tkinter）

## 项目介绍
一款轻量化桌面置顶悬浮背单词软件，自动解析PDF词库，提取单词、音标、中文释义；支持平滑滑动切换、自动轮播、窗口拖拽，学习网课/办公时随时记忆六级、考研英语单词，无广告、开源免费。

##代码说明
纯AI,0手搓

## 功能特性
1. 窗口置顶悬浮，可自由拖动，调节透明度
2. 自动解析PDF单词文档，正则分离单词/音标/释义，自动去重
3. 左右滑动动画切换单词，切换流畅不生硬
4. 5秒自动轮播单词，支持一键暂停/继续
5. 手动上一个/下一个单词按钮，底部显示进度计数
6. 右键菜单：重载PDF、暂停滚动、退出程序
7. 无PDF文件时内置默认六级单词示例兜底
8. 支持使用PyInstaller打包为独立EXE，免Python环境运行
